# Netflix EDA & Insights Analysis

## Project Overview
Exploratory Data Analysis on Netflix Movies & TV Shows dataset to understand content trends, genres, and production patterns.

## Dataset
- **Source:** Kaggle (netflix-shows)
- **Rows:** 8,807 titles
- **Columns:** 12 (title, type, country, rating, genre, etc.)

## Analysis Highlights
- 72% Movies vs 28% TV Shows
- USA leads with 2,500+ titles
- International Movies is the top genre
- TV-MA is the most common rating

## Key Insights
1. Netflix prioritizes movies over TV shows for viewer variety
2. Netflix is heavily US-centric despite global operations
3. COVID-19 pandemic impacted content production in 2020
4. International content is critical to Netflix's global strategy
5. Netflix targets mature adult audiences (TV-MA)

- Matplotlib, Seaborn
- Google Colab
- Kaggle (dataset source)
